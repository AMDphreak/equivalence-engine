= Qt Upgrader (D version)
:revdate: 2026-03-21
:toc: left

A high-performance, compile-time adaptation toolkit for automatically upgrading Qt5 code to Qt6.

== Features

* **Non-Bloated**: Compiled D binary (no Python runtime required).
* **SDL Rules**: Uses human-friendly SDL rulesets.
* **Fast**: Optimized regex processing.
* **CI Ready**: Easily integrated into GitHub Actions.

== Installation

You only need the `dub` package manager (part of the D compiler suite).

[source,powershell]
----
git clone https://github.com/AMDphreak/qt-upgrader.git
cd qt-upgrader
dub build -b release
----

The binary will be available in the root folder as `qt-upgrader` (or `qt-upgrader.exe`).

== Usage

[source,powershell]
----
./qt-upgrader --path "C:/my-project" --rules "rules/qt5_to_qt6.sdl"
----

=== Options

* `-p, --path`: Path to process (folder or file).
* `-r, --rules`: Path to the SDL rules file.
* `-e, --extensions`: Comma-separated list of extensions (default: `.py,.cpp,.h`).
* `-d, --dry-run`: See changes without applying them.

== Ruleset (SDL)

Rules are defined in `rules/qt5_to_qt6.sdl`. Example:

[source,sdl]
----
ruleset "MyRules" {
    replace "OldName" "NewName"
    regex "\\.oldFunc\\(\\)" ".newFunc()"
}
----
